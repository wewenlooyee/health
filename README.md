# 🩺 智慧健康日誌與風險評估系統

## 專案簡介
本專案為一個全端健康數據追蹤應用程式。使用者可記錄每日睡眠時數、步數與心情分數。系統會透過後端實作之「多層分支決策樹演算法」自動評估使用者的健康風險等級，並於前端提供視覺化的圖表分析與智慧貼心話回饋。

## 技術框架
* 前端: HTML5, CSS3, JavaScript (Chart.js)
* 後端: Node.js, Express.js
* 資料庫: SQLite3

## 核心功能與亮點
1. 多層決策樹風險評估：依序綜合判定睡眠、活動量與心情。
2. 動態主題與深色模式：支援多種色彩主題與 Dark Mode。
3. 混合式視覺化圖表：整合長條圖與折線圖，並具備風險自動標色功能。
4. 即時健康儀表板：自動計算並呈現本週平均健康數據。
5. CSV 報表匯出：一鍵下載所有歷史健康紀錄。
